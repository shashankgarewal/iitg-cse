# IIT Guwahati CSE Courseworks & Projects

This repository serves as a centralized, consolidated archive of some of my computer science courseworks and projects completed during my academic studies at *Indian Institute of Technology, Guwahati (IITG)*. 

Each project has been reconstructed into its respective directory using a clean, linear git history preserving all original commits, author metadata, and chronological commit dates.

---
